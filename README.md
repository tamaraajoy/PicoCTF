## 🧩 Apa itu picoCTF?

picoCTF adalah platform latihan keamanan siber (cybersecurity) yang dikembangkan oleh Carnegie Mellon University. Cocok untuk pemula hingga menengah yang ingin belajar topik-topik seperti:

- 🔐 Kriptografi
- 🧠 Reverse Engineering
- 🕵️ Digital Forensik
- 🌐 Web Exploitation
- 📦 Binary Exploitation
- 📂 Steganografi
- 🧰 Sistem & Scripting
